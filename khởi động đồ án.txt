==================================================================
   HƯỚNG DẪN CÀI ĐẶT VÀ CHẠY ĐỒ ÁN QUẢN LÝ KHO SÁCH
==================================================================

Chào mừng bạn đến với đồ án Quản lý Kho Sách!
Để có thể chạy được chương trình này trên máy của bạn, vui lòng làm theo các bước dưới đây.

------------------------------------------------------------------
PHẦN 1: CÀI ĐẶT CÁC PHẦN MỀM CẦN THIẾT
------------------------------------------------------------------

Bạn sẽ cần cài đặt 2 thứ sau: Python và XAMPP (để có MySQL Server).

1. CÀI ĐẶT PYTHON:
   - Truy cập trang chủ Python: https://www.python.org/downloads/
   - Tải phiên bản Python mới nhất (ví dụ: Python 3.11, 3.12).
   - Khi cài đặt, **hãy nhớ tick vào ô "Add Python to PATH"** ở ngay màn hình cài đặt đầu tiên. Đây là bước rất quan trọng!
   - Cứ nhấn "Install Now" và để quá trình cài đặt hoàn tất.

2. CÀI ĐẶT XAMPP (Cách dễ nhất để có MySQL):
   - Truy cập trang chủ XAMPP: https://www.apachefriends.org/index.html
   - Tải phiên bản XAMPP phù hợp với hệ điều hành của bạn.
   - Chạy file cài đặt và cứ nhấn "Next" cho đến khi hoàn tất. Bạn không cần thay đổi thiết lập gì cả.

------------------------------------------------------------------
PHẦN 2: THIẾT LẬP CƠ SỞ DỮ LIỆU (QUAN TRỌNG NHẤT!)
------------------------------------------------------------------

Đây là bước quan trọng nhất để chương trình có dữ liệu để chạy.
File dữ liệu hoàn chỉnh đã được chuẩn bị sẵn trong thư mục `data` với tên là `database_schema.sql`.

Bạn có thể chọn 1 trong 2 cách sau:

---
**Cách 1: Dùng XAMPP và phpMyAdmin (Khuyên dùng cho người mới)**
---

1.1. KHỞI ĐỘNG XAMPP:
   - Mở "XAMPP Control Panel" (bạn có thể tìm trong Start Menu).
   - Nhấn nút "Start" ở dòng "Apache".
   - Nhấn nút "Start" ở dòng "MySQL".
   - Khi cả hai dòng này chuyển sang màu xanh lá là thành công.

1.2. TRUY CẬP phpMyAdmin:
   - Mở trình duyệt web (Chrome, Firefox,...) và truy cập địa chỉ: http://localhost/phpmyadmin

1.3. IMPORT CƠ SỞ DỮ LIỆU:
   - Tại giao diện phpMyAdmin, thực thi file `database_schema.sql` là đủ. File này đã chứa lệnh tạo CSDL.
   - Nhìn lên thanh menu phía trên và tìm đến tab "Import" (Nhập).
   - Trong trang Import:
     + Nhấn vào nút "Choose File" (Chọn tệp).
     + Một cửa sổ sẽ hiện ra, bạn hãy tìm đến thư mục của đồ án, vào tiếp thư mục `data` và chọn file `database_schema.sql`.
     + Kéo xuống dưới cùng và nhấn nút "Go" (hoặc "Import").
   - Chờ một lát để quá trình import hoàn tất. Nếu bạn thấy thông báo "Import has been successfully finished", xin chúc mừng!

---
**Cách 2: Dùng MySQL Workbench (Nếu bạn đã có sẵn)**
---

2.1. MỞ MySQL WORKBENCH:
   - Mở ứng dụng và kết nối vào server MySQL của bạn.

2.2. CHẠY FILE SCRIPT:
   - Trên thanh menu, chọn "File" -> "Open SQL Script...".
   - Một cửa sổ sẽ hiện ra, bạn hãy tìm đến thư mục của đồ án, vào tiếp thư mục `data` và chọn file `database_schema.sql`.
   - Toàn bộ nội dung file sẽ được mở ra trong một cửa sổ soạn thảo.
   - Nhấn vào biểu tượng tia sét màu vàng (Execute) để thực thi toàn bộ script.

   - Chờ một lát để quá trình thực thi hoàn tất. Bạn có thể xem kết quả ở cửa sổ "Action Output" phía dưới. Nếu không có lỗi màu đỏ nào, xin chúc mừng!

   - Để chắc chắn, bạn có thể nhấn nút "Refresh" ở danh sách Schemas (menu bên trái) và sẽ thấy CSDL `bookwarehousemanagement` xuất hiện với đầy đủ các bảng.


------------------------------------------------------------------
PHẦN 3: CÀI ĐẶT THƯ VIỆN VÀ CHẠY ỨNG DỤNG
------------------------------------------------------------------

1. MỞ COMMAND PROMPT (CMD):
   - Mở thư mục gốc của đồ án.
   - Trên thanh địa chỉ của cửa sổ thư mục, gõ `cmd` rồi nhấn Enter. Một cửa sổ dòng lệnh màu đen sẽ hiện ra ngay tại thư mục đó.

2. CÀI ĐẶT CÁC THƯ VIỆN PYTHON:
   - Trong cửa sổ cmd vừa mở, copy và dán dòng lệnh sau rồi nhấn Enter:
     pip install flet flet-core mysql-connector-python pillow reportlab openpyxl

   - Chờ cho quá trình cài đặt hoàn tất.

3. CHẠY ĐỒ ÁN:
   - Vẫn ở trong cửa sổ cmd đó, gõ lệnh sau rồi nhấn Enter:
     python MainFrame.py

   - Nếu mọi thứ thành công, cửa sổ ứng dụng sẽ hiện lên và bạn có thể bắt đầu trải nghiệm!

Chúc bạn có những trải nghiệm tuyệt vời với đồ án!
